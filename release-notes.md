# Release Package

This output folder is prepared for GitHub Pages deployment.

## Included in the release package

- `index.html`
- `styles.css`
- `app.js`
- `config.js`
- `config.example.js`
- `supabase-schema.sql`
- `.nojekyll`

## Before you publish

1. Open `config.js`
2. Add your Supabase project URL
3. Add your Supabase anon key
4. Upload the release files to the root of your GitHub repository
5. Enable GitHub Pages from the `main` branch

## Notes

- Website copy is English-only
- Chat communication can stay in Chinese
- Mobile and desktop layouts are both included
